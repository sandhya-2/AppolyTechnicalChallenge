# AppolyTechnicalChallenge
The AppolyTechnicalChallenge displays a list of Harry Potter characters from the API.

# Installation
Can be used with Xcode 15.1.

# Framework
SwiftUI

# Architecture
This application uses MVVM for the views.

# Design Patterns
Async await.
